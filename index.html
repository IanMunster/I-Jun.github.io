About

Welcome Reader, I'm a Dutch solo game developer. Interested in making fantasy games, adventure/rpg like games. I'm currently focused on 2D games in the GODOT Game Engine. English and Dutch language, but prefer English as the default language in projects.

On this page you can read some More about me, See my mayor Skills and Coding Languages qualifications, See some of the Games and Projects I worked on, And find information on how to Contact me!
